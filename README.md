# Project-1
Modern Admin Dashboard with Secure Login
A responsive and professional admin panel built with PHP, HTML, and CSS, featuring a secure login system and a modern dashboard. This project demonstrates session handling, interactive UI elements, and responsive layout design, making it ideal for portfolios or learning modern web development.
Features
Secure Login: Fixed admin credentials with PHP session authentication.
Dashboard: Left sidebar navigation with icons, interactive cards for Users, Performance, Notifications, and Quick Settings.
Profile & Settings: Admin profile display and simulated password update functionality.
Modern Design: Colorful purple-pink gradient theme, hover effects, and glassy cards.
Responsive: Fully usable on mobile, tablet, and desktop.
Technologies Used
Frontend: HTML5, CSS3, Font Awesome Icons
Backend: PHP (Sessions & simple logic)
Design: Modern sidebar layout with grid cards and topbar profile dropdown
